<h1>Motivation</h1>

This applicaiton has been made for one simple reason. To test Typescript, React.js and other web technologies.

<h1>Tech stack</h1>

<ul>
    <li>React.js</li>
    <li>TypeScript</li>
    <li>react-query</li>
    <li>react-context</li>
    <li>react-dnd</li>
</ul>
